# Odin: Restaurant Page

The _"Restaurant Page"_ project is part of the JavaScript Course in The Odin Project's Full Stack JavaScript path. In this project, I was introduced to JavaScript ES6, npm and Webpack.

## Setup & Run Locally

To get a local copy of this project up and running, follow these steps:

### 1. Clone the Repository

```bash
https://github.com/mukudanieru/odin-restaurant-page.git
```

```bash
cd odin-restaurant-page
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Start the Development Server

```bash
npx webpack serve
```
